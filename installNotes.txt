====================================
 SharpCloud Item Syncroniser Widget
========== version 0.99 ============

To install the widget, unzip all the
files on to you computer.

In the \release folder, double click
on the file called:

 SCItemSyncroniser.exe

This will only work on Windows PC's
running .NET 4.5 or later.
